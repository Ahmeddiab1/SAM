Edit Array

The Edit Array window allows you to enter a single column of time series data. The number of rows in the table should be the same as the number of time steps in the simulation, which is usually determined by the number of records in the weather file. For example, for hourly simulations, the weather file would contain 8,760 records (365 days/year × 24 hours/day × 1 time step/hour = 8760 time steps).

Change time step

Click change time step to enter the simulation time step in minutes: 60 for hourly time steps, 15 for 15-minute time steps, etc.

Copy

Click Copy to copy the data in the table to your computer's clipboard. You can then paste the data into a spreadsheet or text file.

Paste

Click Paste to paste time series data from your computer's clipboard into the table. For example, for hourly data, you could copy a column of 8,760 values from a spreadsheet or text file and paste it into SAM.

Import

Click Import to import time series data from a properly formatted text file. The file should have one column of data, with a header in Row 1. SAM ignores the data in Row 1

Export

Click Export to export data from the table to a text file. Be sure to include a file extension (.txt, .csv, etc.) when you save the file.
