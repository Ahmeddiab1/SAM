Battery Storage

The battery storage model allows you to analyze the performance of the following types of batteries:

•Lead-acid

•Lithium-ion

•Vanadium redox flow

•All iron flow

Note. The battery storage model is available with the Detailed Photovoltaic model and the Generic System model. The implementation of the battery model with the Generic System model is intended to model any kind of power system with battery storage when you have a time series generation profile measured from an installed system or generated by SAM or another model. For example, you could model a wind system with storage by running the wind power model in SAM, exporting the wind system's generation profile to a text file, and importing the data to SAM's Generic System model.

The model accounts for important factors contributing to battery performance, including voltage variation with charge state, capacity fade with cycling and age, and temperature effects on capacity. The model provides a realistic picture of how the battery performs and degrades over time. As with other models in SAM, the battery model inputs strike a balance between simplicity and complexity to be usable by people with varying degrees of interest and expertise. Many of the inputs are parameters that are available on typical manufacturer data sheets. For other inputs, you can apply reasonable default parameters by choosing a battery type.
